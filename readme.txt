== Description ==

Book has been added by Admin and from Front end we can search books by Name, author, publisher and ratings.

Books have own detail page and display all related details to bike details page.

== Instruction to Install plugin ==

1. Download zip file

2. Extract zip file to plugin folder

3. And added extracted file to wp-content folder

4. Install plugin through Admin backend

5. Add shortcde to front end ([book-search-library])

6. See book list in front end.

7. Search books by Title, author, publisher, price and rating.

Thank you for using this plugin.
